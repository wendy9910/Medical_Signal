### EX03【Measuring Humidity&Temperature】
**Sensor:** DHT11 <br />
**<基本資料>** <br />
供電電壓:3.3-5.5V <br />
輸出:單總線數字信號 <br />
測量範圍:濕度 20-90%RH，溫度 0-50度 <br />
測量精度:濕度 +-5%RH，溫度+-2度 <br />
分辨率:濕度 1%RH，溫度 1度 <br />
長期穩定性: <+-%RH/年 <br />
